# 🎓 School Management System

A modern **School Management System** built using PHP and MySQL to efficiently manage school operations such as student records, teacher data, attendance, results, and more.

This project is designed for schools and small institutes to **streamline administrative tasks, improve communication, and manage academic data effectively**.

---

## 🚀 Features

* 📚 Student Record Management
* 👨‍🏫 Teacher Management
* 📝 Attendance Management
* 📢 Notice & Announcement System
* 📊 Exam Result Upload & Tracking
* 📂 Notes & Study Material Upload
* 🚌 Bus Service Management
* 📅 Timetable & Syllabus Management
* 🔐 Secure Login System
* 🌙 Dark Mode Support
* 🌍 Multi-language Support

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Backend:** PHP

## ⚙️ Installation & Setup

### Step 1: Setup Environment

* Install XAMPP (or any PHP + MySQL server)
* Start **Apache** and **MySQL**

### Step 2: Database Setup

* Open **phpMyAdmin**
* Create database: `_sms`
* Import file: `database/_sms.sql`

### Step 3: Project Setup

* Copy project folder into:

  ```
  C:\xampp\htdocs\
  ```
* Make sure structure is:

  ```
  htdocs/school-management-system/
  ```

### Step 4: Run Project

Open browser and go to:

```
http://localhost/school-management-system
```
